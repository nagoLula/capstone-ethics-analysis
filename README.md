# Capstone Ethics Analysis: SLP Autonomy vs Aspiration Risk

## Research Question
What is the relationship between licensed Speech-Language Pathologists’ years of experience and their negotiation of patient autonomy versus minimizing aspiration risk in Skilled Nursing Facilities?

## Repository Structure
- `data/` – Cleaned survey data and raw Qualtrics export
- `scripts/` – Python scripts for analysis
- `notebooks/` – Jupyter notebooks for interactive exploration
- `outputs/` – Charts and statistical results
- `README.md` – Project documentation

## Methods
- Data cleaning in Excel
- Python analysis in VS Code using:
  - **Descriptive statistics**
  - **Correlation & ANOVA**
  - **Visualizations** (bar chart, boxplot, heatmap)

## Ethics & Compliance
- IRB-approved study
- Data anonymized and stored securely
- Public repo contains only de-identified data
